10 trin. Ingen ventetid. Du uploader jobopslag + SoMe-billeder — systemet klarer resten.

01

Upload jobopslag + SoMe-billeder

Log ind og upload PDF med stillingsbeskrivelsen samt kreative billeder til Meta, LinkedIn og Google Ads-kampagner.

02

Budget Optimizer AI beregner mediamix

AI analyserer historiske data — CTR, CPA, CR per kanal og stillingskategori — og beregner det optimale budgetmix på Meta, LinkedIn og Google.

03

Automatisk distribution til alle kanaler

Opslaget distribueres automatisk til jobportaler. TalentIQs egne Meta, LinkedIn og Google-konti aktiverer kampagnerne med de uploadede SoMe-billeder.

04

Kandidater anonymiseres ved ansøgning

Kandidater lander på TalentIQs ansøgningsside. Persondata anonymiseres straks — CV flyttes til krypteret mappe. GDPR by design fra dag 1.

05

AI CV Screener scorer alle CV'er 1–100

Alle CV'er scores automatisk mod stillingens krav. Faglige kompetencer, erfaring og branchematch. Objektiv rangering — ingen bias. 24/7.

06

80–100 point: direkte booking i lederens kalender

Top-kandidater modtager automatisk invitationsmail med Calendly-link og booker direkte i lederens Outlook/Google kalender.

07

70–79 point: ansættende leder vurderer manuelt

Grænsekandidat flagges til lederens manuelle gennemgang i dashboardet. Godkend → booking. Afvis → afslagsmail + PDF.

08

0–69 point: automatisk afslag + kandidatvurdering PDF

Alle afviste modtager automatisk en professionel afslagsmail med dokumenteret kandidatvurdering PDF — hvad der matchede og hvad der manglede.

09

Kandidatdata pushes til kundens ATS

Alle fremrykkede kandidater pushes automatisk via REST API — Teamtailor, HR-Manager, Greenhouse, Personio. Kunden forlader aldrig sit eget system.

10

Al kampagnedata ind i TalentIQ → næste kampagne optimeres

CTR, CPA, CR1 og CR2 fra alle kanaler samles i realtidsdashboard. Budget Optimizer AI lærer og forbedrer næste kampagnes mediamix automatisk.

1 · Kreativ-generering
Bannerbear API
⚠ Mangler API-nøgle
Genererer automatisk Meta- og LinkedIn-annoncedesigns i korrekt format (1:1, 4:5, 16:9) med joblogo, tekst og brandfarver. Forhindrer afvisning fra Meta pga. tekst-overflow eller forkert format-ratio.
Make.com flow:
Stripe betaling → Parse jobopslag → Bannerbear API → Generér 3 formater → Meta/LinkedIn upload
2 · Budget Trigger
Stripe → Meta Ads API
⚠ Mangler
Tildeler automatisk det beregnede annonce-budget til en unik kampagne i Meta Ads Manager ved betaling. Uden dette skal du manuelt aktivere hver kampagne i Meta's dashboard — ca. 15 min. per job.
Make.com flow:
Stripe webhook → Beregn budget split (Meta/LI/Google) → Meta Marketing API → Opret kampagne → Set budget → Aktivér
3 · Kalender-validering
Calendar Sync Health
⚠ Ingen kalendre tilknyttet
Tjekker dagligt om lederens Outlook/Google-kalender har ledige tider i kommende uge. Sender automatisk SMS hvis ingen åbninger: "Top-kandidater venter — din kalender er tom". Forhindrer at gode kandidater venter i tomrum.
Daglig tjek (kl. 08:00):
Outlook OAuth → Hent kalender → Ledig tid næste 7 dage? → Ja: fortsæt · Nej: SMS til leder via Twilio
Tilknyttede ledere:
Ingen endnu — tilføj nedenfor
4 · PDF-generering
PDFMonkey API
⚠ Mangler API-nøgle
AI'en genererer kandidatvurderingstekst — PDFMonkey indsætter den i TalentIQ's brandede skabelon og returnerer en færdig PDF der vedhæftes afslagsmailen automatisk. Kandidaten modtager professionel feedback på sekunder.
Make.com flow:
AI score < 70 → Claude API (vurderingstekst) → PDFMonkey (PDF) → Resend (afslagsmail + PDF vedhæft)
5 · API Fejlhåndtering
Retry-loop + Fallback-mail + Opsgenie
⚠ Ingen retry-logik aktiv
Retry-loop (Make.com)
Hvis ATS API fejler: vent 1t → prøv igen × 3 over 24 timer. Logges i fejllog med timestamp og fejlkode.
Fallback-mail
Hvis alle retries fejler: send kandidat-CV direkte til kundens email med note: "Vi kunne ikke pushe til dit ATS — her er kandidaten manuelt".
Opsgenie / SMS-alarm
Kritiske fejl sender SMS til din telefon straks. Du ser det på din Mac / iPhone og kan manuelt intervenere inden kunden opdager det.
Make.com Retry-flow:
ATS push → Fejl? → Vent 60 min → Retry #1 → Fejl? → Vent 4t → Retry #2 → Fejl? → Vent 18t → Retry #3
→ Stadig fejl? → Resend fallback-mail til kunde + Opsgenie SMS til admin → Log i fejllog
